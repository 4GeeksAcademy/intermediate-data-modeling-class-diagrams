Practica guiada de **diagramas de clases UML** y **modelado de datos** con [diagram.4geeks.com](https://diagram.4geeks.com/). Construiras un unico modelo acumulativo de una **plataforma de cursos online** en doce pasos.

Este paquete **no tiene tests automaticos** ni **entrega de archivos**. Trabaja solo en diagram.4geeks.com; usa los README de cada paso como guia. Compara tu diagrama final con el `classDiagram` de referencia en el paso **12 - Modelo final**.

## Inicio en Codespaces (recomendado)

1. Abre este repositorio en [GitHub Codespaces](https://github.com/codespaces).
2. Espera a que termine la configuracion del entorno (LearnPack `5.0.348` se instala automaticamente).
3. Ejecuta:

```bash
learnpack start
```

4. Abre [diagram.4geeks.com](https://diagram.4geeks.com/) en el navegador y manten un solo diagrama durante toda la practica.

## Estructura de ejercicios

Cada carpeta de paso incluye solo:

- `README.md` — instrucciones en ingles
- `README.es.md` — instrucciones en espanol

No hay archivos de respuesta, tests ni rubricas en este repositorio.

## Orden sugerido

Los pasos `01` a `12` se apoyan entre si. Puedes saltar pasos en LearnPack, pero los pasos posteriores asumen que ya modelaste los conceptos anteriores.

## Modelado en diagram.4geeks.com

Usa **cajas de clase**, **propiedades tipadas** y **etiquetas de texto en los conectores** para cardinalidad (`1`, `N`, `*`). Si la interfaz no ofrece una funcion UML (herencia, enums, rombos de composicion), usa una **propiedad string** o una **nota en el enlace**.
