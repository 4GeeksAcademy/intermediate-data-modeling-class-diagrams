# 03 - Entidad Instructor

Version en ingles: [README.md](./README.md)

## Escenario

Cada curso lo imparte un perfil de instructor guardado en el sistema.

## Instrucciones

Agrega la clase **`Instructor`** con:

| Propiedad | Tipo |
|-----------|------|
| `id` | `int` |
| `name` | `string` |
| `bio` | `string` |

Mantén `Course` y `Student` sin cambios. Sin relaciones aun.

## Lista de verificacion

- [ ] `Instructor` existe con campos tipados.
- [ ] Tres clases visibles: `Course`, `Student`, `Instructor`.
