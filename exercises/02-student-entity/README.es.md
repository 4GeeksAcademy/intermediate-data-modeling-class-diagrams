# 02 - Entidad Student

Version en ingles: [README.md](./README.md)

## Escenario

Los alumnos se registran en la plataforma. Agregalos como entidad separada.

## Instrucciones

En el **mismo diagrama**, agrega la clase **`Student`** con:

| Propiedad | Tipo |
|-----------|------|
| `id` | `int` |
| `name` | `string` |
| `email` | `string` |

Aun no enlaces `Student` con `Course`.

## Lista de verificacion

- [ ] `Course` del paso 01 sigue presente.
- [ ] `Student` tiene tres propiedades tipadas.
